# 🤖 AI Text Summarizer

AI-powered Text Summarizer built using Python, Streamlit, and Hugging Face Transformers.

## Features

- Summarize long text instantly
- Short / Medium / Long summary options
- Character and word counter
- Copy summary feature
- Download summary as text file
- Responsive UI

## Tech Stack

- Python
- Streamlit
- Hugging Face Transformers
- NLP
- Git & GitHub

## Installation

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
